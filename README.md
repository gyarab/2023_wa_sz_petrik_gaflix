# 2023_wa_sz_petrik_gaflix

### Create venv
- `py -3 -m venv venv`

### Activation
1. `source ./venv/Scripts/activate`
2. `pip install -r requirements.txt`

### Run server
- `python manage.py runserver`

### Migrations
1. `python manage.py makemigrations`
2. `python manage.py migrate`

#### Create admin
- `./manage.py createsuperuser`
